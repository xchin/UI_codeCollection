Alerts

Basic Alert

Informational message content.

Code

						<div class="alert">
							<p class="alert-text">
								Text
							</p>
						</div>

Page Level Alerts

Help message content with provided icon.

Success message content with provided icon.

Warning message content with provided icon.

Critical message content with provided icon.

Inline Alerts

Help message content with provided icon.

Success message content with provided icon.

Warning message content with provided icon.

Critical message content with provided icon.

Code

						<div class="{alert|alert_inline} [alert-help|alert-success|alert-warning|alert-critical]">
							<p class="alert-text">
								Text
							</p>
						</div>

Custom Icon

Informational message content with custom icon. These alerts can also contain a long string of text without breaking the alignment.

Code

						<div class="alert">
							<span className="alert-icon icon icon-small <your-icon>"aria-hidden="true"></span>
								<p class="alert-text">
									Text
								</p>
						</div>
