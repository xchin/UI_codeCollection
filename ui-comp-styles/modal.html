Close options

  1. The base class modal-dismiss_trigger applies only the close behavior.
  2. The class modal-dismiss_x-trigger is the class that applies behavior and the "X" styling.
  3. The class modal-dismiss_x only applies the "X" styling without a close behavior.

Code

// Modal trigger: needs to pass the ID of the modal it controls
<button class="btn modal-trigger btn" data-modal-id="_demoModal">Trigger CTA</button>
// Modal component: needs to match the ID of the trigger
<div class="{modal|modal-glimpse|modal-flow}" id="_demoModal">
	<div class="modal-wrapper">
		<button class="{modal-dismiss_trigger|modal-dismiss_x-trigger|modal-dismiss_x}" data-modal-id="_demoModal">
			<span class="a11yText">Close</span>
		</button>
		<div class="modal-content">
			<header class="modal-header"><h2>This is a header</h2></header>
			<div class="modal-body">Lorem ipsum ... </div>
		</div>
	</div>
</div>
