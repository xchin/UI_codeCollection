Block Buttons (full width)

Code

<button className="bsBtnMod bsBtnMod-block">Block button</button>

Button Groups:

Code

<div class="bsBtnMod-group">
	<!-- buttons of same size -->
</div>

Two-up Buttons:

This is a very common pattern for having buttons side-by-side. Just ensure your minifying your HTML, or you may get a strange carriage return after the first button. NOTE: the class blocks-for-mobile ensures that at narrow widths, the buttons become block (full-width).

Code

<div class="bsBtnMod-twoUp [blocks-for-mobile]">
	<!-- buttons of same size -->
</div>

<button className="btn bsBtnMod-link">Button as a link</button>
