Refresh Page

Example code below:

<body class="addFlowTransition" id="control ">
	<a href="some/other/page.html" class="flowTrigger">Trigger Flow</a>
</body>

// To reverse the transition, add the hasFadeTransition class for the component to remove it after load

</body>
