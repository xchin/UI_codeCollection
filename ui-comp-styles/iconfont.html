Instructions:

The following is the syntax for using the icon font:

<element class="icon icon-{size} {classname}" aria-hidden="true"></element>
  1. The first class, icon, assigns all the generic icon properties to the container element.
  2. The second class, icon-{size}, assigns the correct sizing to the icon. The choice is small, medium and large.
  3. The third class, {classname}, assigns the correct unicode glyph to the psuedoelement
  4. The aria-hidden="true" ensures that screenreaders do not read the glyph aloud.

Please note: this element should remain empty; don't add additional text or child elements.

Example one: inline version

<p><span class="icon icon-small icon-critical-small" aria-hidden="true" ></span> Your email and password combination failed.</p>

Example two: stacked version

<div class="icon icon-large icon-critical-large" aria-hidden="true"></div>
<p>Your email and password combination failed.</p>
