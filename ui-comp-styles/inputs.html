Inputs and Selects:

Simple Input

An alternative class of 'form-group_hiddenLabel' creates a label-less accessible input. Labels are preferred for accessibility.

Code

								<div class="{form-group|form-group_hiddenLabel} [has-error]">
								<label for="form-control-error" id="simple-form-control">Text</label>
								<input class="form-control form-control _demo-input"
								type="text" id="simple-form-control" />
								</div>

Complex Input

These inputs are intended to mimic inputs with variable elements "inside". Of course, an actual <input> element cannot have child elements, so we have to mimic it with the following modifications:

  • Same parent element as above, but rather than a direct <input> child, use a <div> with a class of form-control-complex .
  • Within the above element, a classless <input> is placed.

Placing icons "within" the input is done by placing a <span> within the form-control-complex element referencing your desired icon. Most likely, this will be an error icon that can be placed preemptively and shown as needed.

Basic Example

Code

								<div class="form-group [has-error]">
								<label for="form-control_complex">Text</label>
								<div class="form-control form-control_complex">
								[<span class="form-control-icon icon icon-small <your-icon>"></span>]
								<input type="text" id="form-control_complex" />
								<span class="form-control-error-icon icon icon-small icon-critical-small"></span>
								</div>
								</div>

Inputs with messaging: help or error

Many times, we need to communicate information to the user about a particular input. This can be done easily by ...

  • Add has-message to the parent container: form-group
  • Append a simple <span class="form-control-message"> appended to the parent container above.

Simple Input with Message

I'm helpful!

Simple Input with Error Message

I'm sorry, but this is invalid!

Complex Input with Message

I'm helpful too!

Code

								<div class="form-group has-message [has-error-with-message]">
								<label for="form-control-with-message">Text</label>
								<input class="form-control" type="text"
									id="form-control-with-message" />
								<span class="form-control-message">Text</span>
								</div>

Selects (Version One)

Code

								<div class="form-group [has-error]">
								<label for="select">Text</label>
								<select class="form-control" id="select">
								<option value="">text</option>
								</select>
								[<span class="selectIcon_error"></span>]
								</div>
